本書說明

南明野史（又名《明末五小史》、《五藩實錄》、《明末紀事補遺》）
版本：
　　據涵芬樓舊藏稿本「釐訂」，民國十九（１９３０）年商務印書館排版印行。三卷及附錄一卷。
　　本書排版後，復獲《明季五藩實錄》（又名《明末五小史》）一書，始知即為未經王鍾麒氏「釐訂」前所印行者。《實錄》分五篇：一曰福藩、二曰唐藩、三曰唐王聿〈金粵〉、四曰魯藩、五曰桂藩（各篇又有分上下二篇者）。論理，原應印行未「釐訂」前之版本，但以排校既竣，乃決定即以本書印行。
　　本書與《實錄》比校，除改動卷次與篇目外，清歷紀年均改為明歷紀年。原有「王師」、「我朝」一類字樣亦改為「清師」、「清朝」，「福王」、「唐王」、「桂王」等改為「安宗皇帝」、「紹宗皇帝」、「永歷皇帝」（簡稱為「帝」）。另，「自序」刪削頗多。以上為兩書主要相異之處。
作者：
　　原作者南沙三余氏，生平不詳，成書於清乾隆四（１７３９）年；民國十八（１９２９）年由吳縣王鍾麒釐訂。
內容：
　　記載明朝宗室自李自成攻破北京、滿清佔據中原後，先後於南方建立弘光政權、隆武政權、魯王監國、紹武政權及永歷政權前後共十八年的野史。
　　又，李慈銘《受禮廬日記．上集》記《五藩實錄》有云：『其書蕪穢尨雜，全無體裁。然頗有他書所未見者，於唐藩事尤詳，多載其書詔批答；蓋以出思文手制而特存之』。實則「唐藩」事，多本諸閩人《思文大紀》一書。
